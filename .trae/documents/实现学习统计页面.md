# 实现学习统计页面

## 功能需求
1. 开始/停止按钮记录学习答题时间
2. 输入框用于填写总题数和错误题数
3. 自动计算正确率
4. 折线图展示时间和正确率的统计数据
5. 本地存储历史数据

## 实现方案

### 1. 添加导航链接
- 修改 `.vuepress/config.js`，在导航栏添加学习统计页面链接

### 2. 创建统计页面组件
- 在 `.vuepress/theme/components/` 目录下创建 `StudyStats.vue` 组件
- 实现开始/停止时间记录功能
- 实现题数和错误数输入表单
- 实现正确率计算逻辑

### 3. 添加图表库
- 在 `package.json` 中添加 Chart.js 依赖
- 使用 Chart.js 绘制折线图

### 4. 实现数据存储
- 使用 localStorage 存储历史学习数据
- 实现数据的增删查改功能

### 5. 样式设计
- 采用与现有主题一致的风格
- 响应式设计，适配移动端

## 技术栈
- Vue.js 2.x
- Chart.js
- LocalStorage

## 预期效果
- 导航栏新增"学习统计"入口
- 页面包含时间记录器、题数表单和统计图表
- 数据持久化存储，刷新页面后数据不丢失
- 直观的折线图展示学习趋势
